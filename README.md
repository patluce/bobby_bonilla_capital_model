# Bobby Bonilla Day Capital Decision Calculator

This folder is ready for GitHub Pages.

## Files

- `index.html` — the live web-based calculator. No build step and no CDN dependencies.
- `bobby_bonilla_capital_model.xlsx` — downloadable Excel model.
- `.nojekyll` — tells GitHub Pages to serve the files directly.

## Deploy

1. Upload these files to the root of the GitHub repository.
2. Go to **Settings > Pages**.
3. Under **Build and deployment**, select **Deploy from a branch**.
4. Set branch to `main` and folder to `/ (root)`.
5. Save.

The site will publish at:

`https://patluce.github.io/bobby_bonilla_capital_model/`
